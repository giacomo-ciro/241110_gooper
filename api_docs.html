Gooper API Documentation

This API's endpoints allow you to interact with Gooper and its underlying database.

/generate

Generates the model's response to a user's prompt.

Method: POST

Body: it expects a JSON payload in the request body

{
    "prompt": "Your question or prompt here"
}

Response:

{
    "response": "Generated response for: Your question or prompt here"
}

Examples:

cURL

curl -X POST http://brimax.pythonanywhere.com/generate \
     -H "Content-Type: application/json" \
     -d '{"prompt": "Your question or prompt here"}'

Python

import requests

url = 'http://brimax.pythonanywhere.com/generate'
headers = {'Content-Type': 'application/json'}
data = {"prompt": "Your question or prompt here"}
response = requests.post(url, json=data, headers=headers)
print(response.json())

Errors:

If the input is invalid (e.g., missing the prompt field), the API will return a 400 Bad Request error with the following response:

{
    "error": "Invalid input, \"prompt\" is required"
}

/model_version

Returns the version of the currently running model.

Method: GET

Body: no body required

Response:

{
"response": "model_version (e.g. 1.1.2)"
}

Examples:

cURL

curl -X POST http://brimax.pythonanywhere.com/model_version

Python

import requests

url = 'http://brimax.pythonanywhere.com/model_version'
response = requests.post(url)
print(response.json())

/influencer_count

Returns the number of influencers in the current database.

Method: GET

Body: no body required

Response:

{
"response": int
}

Examples:

cURL

curl -X POST http://brimax.pythonanywhere.com/influencer_count

Python

import requests

url = 'http://brimax.pythonanywhere.com/influencer_count'
response = requests.post(url)
print(response.json())
