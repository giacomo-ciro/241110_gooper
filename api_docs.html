API Endpoint Documentation

Currently this API supports only the /generate endpoint. It allows to send a user prompt and receive a response generated by Gooper v1.

Endpoint

URL: /generate

Method: POST

Request

The API expects a JSON payload in the request body:

{
    "prompt": "Your question or prompt here"
}

Response

The API returns a JSON response containing the generated result:

{
    "response": "Generated response for: Your question or prompt here"
}

Example Request (cURL)

curl -X POST http://127.0.0.1:5000/generate \
     -H "Content-Type: application/json" \
     -d '{"prompt": "Your question or prompt here"}'

Example Request (Python)

import requests

url = 'http://127.0.0.1:5000/generate'
data = {"prompt": "Your question or prompt here"}
response = requests.post(url, json=data)
print(response.json())

Error Handling

If the input is invalid (e.g., missing the prompt field), the API will return a 400 Bad Request error with the following response:

{
    "error": "Invalid input, \"prompt\" is required"
}
